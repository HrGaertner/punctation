We present a tool for automatic punctuation. Speak (or write) some sentences into the Mircophon (the textarea). with this you are training our tool. As more user our tool gets as better the results get. The ANN (artifical neural network) will set points and kommas automatical after the end of this project. The data which was inputed will be processed and saved as grammatical patterns during this process the meaning gets lost.
